# PASO 3: Crear carpeta prisma/ y subir schema.prisma

## Método 1: Usando "Create new file" (Más rápido)

1. **Haz clic en "Add file"** → **"Create new file"** (arriba a la derecha)

2. **En el campo de nombre**, escribe exactamente:
   ```
   prisma/schema.prisma
   ```
   ⚠️ IMPORTANTE: El `/` después de `prisma` crea la carpeta automáticamente

3. **NO escribas nada** en el editor de contenido todavía

4. Haz clic en **"Cancel"** (arriba a la izquierda)
   - Esto crea la carpeta `prisma/` vacía

5. Ahora haz clic en **"Add file"** → **"Upload files"**

6. **Arrastra** el archivo desde tu computadora:
   - Ubicación: `C:\Users\user\Desktop\WHATS\prisma\schema.prisma`
   - O haz clic en **"choose your files"** y selecciona el archivo

7. **Desplázate hacia abajo** y escribe el mensaje de commit:
   ```
   Agregar prisma/schema.prisma en ubicación correcta
   ```

8. Selecciona: **"Commit directly to the main branch"**

9. Haz clic en **"Commit changes"** (botón verde)

---

## Método 2: Directamente desde tu carpeta local (Más fácil)

1. Abre la carpeta: `C:\Users\user\Desktop\WHATS\prisma\`

2. En GitHub, haz clic en **"Add file"** → **"Upload files"**

3. **Arrastra el archivo `schema.prisma`** directamente al área de arrastrar de GitHub

4. **IMPORTANTE**: Antes de hacer commit, verifica que en el campo de nombre del archivo aparezca:
   ```
   prisma/schema.prisma
   ```
   Si aparece solo `schema.prisma`, cámbialo manualmente a `prisma/schema.prisma`

5. Commit: `"Agregar prisma/schema.prisma"`

6. Haz clic en **"Commit changes"**

---

✅ **Cuando termines este paso, avísame y seguimos con el PASO 4**
